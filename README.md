## What is this repo?

This is my project submission for the Digital Literacy course (CSE0001) 
at VIT Bhopal. The project has 5 tasks covering building an online 
presence, professional communication, exploring coding platforms and 
understanding cybercrime. Each task has its own folder with the

## Task Summaries

**Task 1 – Digital Literacy Infographic**
Created a one page infographic titled "Digital Literacy for Students" 
using Canva. It covers what digital literacy means, safe internet 
practices, professional online presence and useful digital tools 
for students.

**Task 2 – Student Digital Portfolio**
Set up profiles on GitHub, LinkedIn and Kaggle. Added education 
details, a GitHub profile README and took screenshots of all 
three profiles as proof.

**Task 3 – Coding and Collaboration Platforms**
Completed beginner Python challenges on HackerRank and built a 
5 question Digital Literacy Awareness Quiz on Google Forms linked 
to a Google Sheet for response collection.

**Task 4 – Professional Email and Etiquette**
Drafted two professional emails — one requesting an assignment 
extension and one expressing interest in a summer internship. 
Also created a Social Media Do's and Don'ts checklist for 
college students.

**Task 5 – Cybercrime Awareness**
Researched UPI/Online Payment Fraud and wrote a 220 word case 
study describing how the scam works and who it targets. Also 
created a Stay Safe Online checklist with 10 actionable tips 
for college students in India.
